# 故障汇总清单

## 所有提示语可配置
 - 现在生成式环境的提示内容都写在后端脚本里了，需要拆解出来，在后端目录下建一个文件夹，存放json文件。执行脚本时读取文件内容作为提示词来执行。对于生图，生视频的脚本则放在json目录的子目录中。目录结构应为json_file/text或img或video。text存默认的分析提示词，img存生成的角色提示词（前端可修改），video存视频生成提示词，即分镜提示内容。后续会根据使用习惯给出接口的需求。

## 拆解剧本方案修改
 - 现在是分块+限制上下文的方式，将文稿输入给llm来分析。这样有个问题：章节间存在漏字多字的情况，即一块不一定是完整的一章。如果我想按照章节分块，那需要有一个结束语，即识别pdf或者word的时候识别到类似"chapter"这样的词的时候进行分块。做一个分块识别词，且在页面可配置。
