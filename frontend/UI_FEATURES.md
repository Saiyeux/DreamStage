# 🎨 UI交互特性说明

## 原神风格沉浸式展示

受原神官网角色展示页面启发，我们实现了更加精美的交互体验。

### ✨ 角色展示 (CharacterShowcase)

**触发方式**: 在剧本分析页面点击任何角色卡片

**特性**:
- 🖼️ **全屏沉浸式体验** - 黑色背景，超大角色形象
- 📋 **左侧信息面板** - 角色名称、基本信息、快速导航
- 📖 **右侧详情面板** - 外貌特征、性格特点、服装风格
- 🎯 **底部角色选择器** - 横向滚动，快速切换角色
- ⌨️ **键盘导航** - 左右箭头切换，ESC关闭
- 🎬 **流畅动画** - 600ms过渡动画，面板从两侧滑入

**操作指南**:
```
← → 方向键   - 切换角色
ESC         - 关闭查看器
鼠标点击     - 底部缩略图快速跳转
```

### 🎬 场景展示 (SceneShowcase)

**触发方式**: 在剧本分析页面点击任何场景卡片

**特性**:
- 🌄 **全屏场景图背景** - 真实场景图作为背景，渐变遮罩
- 📝 **左侧场景信息** - 场景名称、环境描述、镜头设置
- 👥 **右侧角色动作** - 出场角色列表、对白内容
- 🎞️ **底部场景选择器** - 缩略图轮播，快速浏览所有场景
- 🔄 **进度指示器** - 底部显示当前场景进度
- 📸 **图片加载提示** - 加载大图时显示loading动画

**操作指南**:
```
← → 方向键   - 切换场景
ESC         - 关闭查看器
鼠标点击     - 底部缩略图快速跳转
```

## 🎯 设计亮点

### 1. **毛玻璃效果 (Glassmorphism)**
```css
backdrop-blur-lg     /* 背景模糊 */
bg-white/10          /* 半透明白色 */
border border-white/20  /* 半透明边框 */
```

### 2. **渐变遮罩**
- 背景图片顶部和底部添加黑色渐变
- 确保文字可读性
- 营造电影感氛围

### 3. **过渡动画**
- **入场动画**: 面板从左右两侧滑入
- **切换动画**: fade + scale效果
- **悬停动画**: 卡片缩放、阴影增强

### 4. **响应式交互**
- 左右切换按钮固定在两侧
- 底部选择器自动居中
- 移动端友好的触摸操作

## 📱 交互细节

### 角色卡片悬停效果
```
默认状态 → 悬停状态
scale-100 → scale-105
opacity-100 → shadow-2xl
border-transparent → border-purple-200
```

### 场景卡片
- **缩略图**: 图片悬停时放大1.1倍
- **信息面板**: 渐变背景，清晰可读
- **状态标签**: 时间、时长信息以徽章形式展示

### 进度指示器
```
当前场景: w-8 bg-white
其他场景: w-1 bg-white/30
```

## 🎨 配色方案

### 主题色
- **紫色系**: `#667eea` → `#764ba2`
- **蓝色系**: `#3b82f6` → `#6366f1`
- **粉色系**: `#ec4899` → `#f43f5e`

### 半透明层级
- `bg-black/95` - 主背景
- `bg-black/30` - 信息面板
- `bg-white/10` - 交互元素
- `border-white/20` - 边框

## 🚀 性能优化

### 1. **图片懒加载**
- 使用`onLoad`事件追踪加载状态
- 显示loading动画避免空白

### 2. **动画防抖**
- 使用`isAnimating`状态防止快速切换
- 600ms动画延迟确保流畅

### 3. **滚动优化**
```css
.scrollbar-hide {
  scroll-behavior: smooth;
  -ms-overflow-style: none;
  scrollbar-width: none;
}
```
